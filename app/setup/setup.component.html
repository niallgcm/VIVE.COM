JUST GOT THE BOX?
SET UP YOUR VIVE

Vive Setup will walk you through setting up your Vive and installing all of the software that you need to use it. You'll get the:

  • Vive setup tutorial
  • Vive desktop app where you can browse, purchase, download and launch Viveport and other VR experiences
  • Vive Home VR app, your own customizable VR environment
  • Steam Client if not already installed

Requirements: Microsoft .NET Framework 4.6 or later, internet connection, mounting tools (drill, screwdriver etc.)
